# Future Letter 💌

> **"지금의 고민에, 10년 후의 내가 답장을 보낸다면?"**
> 기록과 위로를 연결하는 감성 AI 편지 서비스

## 📮 프로젝트 소개

**Future Letter**는 사용자가 현재의 고민과 감정을 적으면,
AI가 **"10년 후의 나"** 관점으로 따뜻한 편지를 생성해주는 웹 서비스입니다.

단순 생성형 응답이 아니라,
- 지금의 감정을 정리하고
- 실행 가능한 작은 행동을 제안받고
- 편지를 타임캡슐로 저장해
나중에 다시 읽으며 자신의 변화를 확인할 수 있도록 설계했습니다.

---

## 🎯 핵심 가치

- **감정 정리**: 머릿속 고민을 글로 구조화해 현재 상태를 선명하게 인식
- **미래 관점 조언**: "10년 후의 나" 시점으로 불안을 낮추는 공감형 답장
- **기록의 축적**: 고민 + 답장을 타임캡슐로 저장해 회고 가능
- **가벼운 사용성**: 모바일에서도 바로 입력/생성/공유 가능한 단순 UX

---

## ✨ 핵심 기능

### 1) 고민 작성
내 고민을 차근차근 적는 단계
├─ 제목, 내용, 기분은 꼭 입력해야 해요
├─ 빈칸이 있으면 보내기 버튼이 비활성화돼요
├─ 작성 중 내용은 자동으로 임시저장돼요
└─ 페이지를 다시 열어도 이전 내용이 복원돼요


### 2) AI 편지 생성
서버에 요청을 보내 AI 편지를 받아와요
├─ OpenAI 또는 Azure OpenAI로 편지를 생성해요
├─ 공감 → 현실적인 조언 → 응원 순서로 답장을 만들어요
├─ 봉투 애니메이션 뒤에 편지가 화면에 보여요
└─ 응답이 늦으면 서버에서 다시 시도해요

### 3) 결과 액션
받은 편지를 바로 활용할 수 있어요
├─ 저장하기: 타임캡슐 목록에 추가돼요
├─ 공유하기: 공유 기능이 안 되면 자동으로 복사 방식으로 전환돼요
└─ 다시 생성: 같은 고민으로 다른 톤의 편지를 다시 받아요


### 4) 타임캡슐
저장한 고민과 답장을 다시 보는 공간이에요
├─ 최신 기록이 위로 오도록 정렬돼요
├─ 날짜와 당시 기분을 함께 보여줘요
├─ 항목을 누르면 고민 내용과 답장을 자세히 볼 수 있어요
└─ 이전 버전 데이터도 자동으로 맞춰서 읽을 수 있어요
