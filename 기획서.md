# Future Letter (퓨처 레터) 쉬운 기획서

## 1. 서비스 한 줄 소개
"10년 후의 내가 보내는 편지"를 AI로 만들어, 지금의 고민을 위로받고 저장해 두는 감성 웹 서비스

---

## 2. 왜 만드는가
- 사람들은 고민이 많을 때 시야가 좁아진다.
- 지금 감정을 기록하고 나중에 다시 보는 경험이 필요하다.
- Future Letter는 "미래의 나" 시점으로 따뜻한 답장을 주어 마음을 정리하게 돕는다.

---

## 3. 핵심 사용자
- 20대~30대
- 진로, 취업, 관계, 번아웃 고민이 있는 사람
- 감정 기록과 자기 성찰을 원하는 사람

대표 예시
- 취업 준비생: "내 길이 맞는지 모르겠어요"
- 직장인: "번아웃이라 쉬어야 할지, 이직해야 할지 고민돼요"

---

## 4. 핵심 기능 

### 4.1 고민 작성
기능 목적
- 사용자가 지금 마음 상태를 솔직하게 정리하도록 돕는다.

입력 항목
- 제목: 고민을 한 줄로 요약
- 고민 내용: 현재 상황, 힘든 점, 원하는 방향 작성
- 현재 기분: 예 불안, 지침, 설렘, 슬픔, 평온
- 미래에게 질문: "10년 후의 나는 이 문제를 어떻게 봤을까?"

사용 흐름
1. 사용자가 고민 작성 화면에 들어온다.
2. 제목, 고민 내용, 기분, 질문을 입력한다.
3. 미래에게 보내기 버튼을 누른다.

핵심 포인트
- 필수 항목을 채우지 않으면 버튼이 비활성화된다.
- 작성 날짜는 자동 저장된다.
- 작성 중에는 임시저장으로 내용 유실을 줄인다.

기대 효과
- 머릿속 고민을 글로 꺼내며 감정이 정리된다.

### 4.2 AI 편지 받기
기능 목적
- 사용자가 "미래 시점"에서 위로와 조언을 받게 한다.

동작 방식
- 사용자가 입력한 고민과 기분을 바탕으로 AI가 편지를 생성한다.
- AI는 "10년 후의 나" 역할로 답장을 작성한다.

답장 구성
- 인사: 현재의 나를 다정하게 맞이
- 공감: 지금 감정을 이해하고 인정
- 조언: 너무 어렵지 않은 현실적인 제안
- 응원: 스스로를 믿을 수 있게 마무리

사용자 액션
- 저장하기: 이 답장을 타임캡슐에 보관
- 공유하기: 좋은 문장을 외부에 공유
- 다시 생성: 같은 고민으로 새로운 톤의 답장 받기

기대 효과
- 지금의 불안을 줄이고, 한 걸음 행동할 힘을 얻는다.

### 4.3 타임캡슐 저장
기능 목적
- 고민과 답장을 쌓아두고 나중에 다시 볼 수 있게 한다.

저장 내용
- 작성 날짜
- 고민 제목/내용
- 당시 기분
- AI 답장

사용 흐름
1. 답장을 확인한 뒤 저장하기를 누른다.
2. 타임캡슐 목록에 새 기록이 추가된다.
3. 목록에서 항목을 눌러 상세 내용을 다시 읽는다.

핵심 포인트
- 최신순으로 정렬되어 최근 기록을 바로 볼 수 있다.
- 날짜별로 과거 고민을 쉽게 찾아볼 수 있다.

기대 효과
- 감정과 생각의 변화를 시간 축으로 확인할 수 있다.

### 4.4 감정 히스토리
기능 목적
- 사용자의 감정 변화를 한눈에 보여준다.

제공 정보
- 자주 선택한 감정
- 기간별 감정 변화 흐름
- 고민에서 자주 나온 키워드

표시 방식
- 감정 분포: 간단한 막대/원형 그래프
- 감정 추이: 월별 또는 주별 선 그래프
- 키워드: 많이 나온 단어 순서대로 표시

기대 효과
- "요즘 내가 어떤 상태인지"를 객관적으로 이해할 수 있다.

### 4.5 마이페이지
기능 목적
- 나의 기록을 요약해서 자기 성장을 느끼게 한다.

주요 항목
- 총 작성 편지 수
- 최근 작성 활동
- 자주 느낀 감정
- 자주 나온 고민 키워드

사용 흐름
1. 사용자가 마이페이지에 들어온다.
2. 내 기록 요약을 확인한다.
3. 필요한 경우 타임캡슐 상세로 이동해 다시 읽는다.

기대 효과
- "나는 꾸준히 기록하고 있다"는 성취감을 느끼고 재방문 동기가 생긴다.

---

## 5. 화면 구성

### 5.1 메인 화면
- 서비스 소개 문구
- "미래의 나에게 편지 쓰기" 버튼
- 오늘의 짧은 명언

### 5.2 고민 작성 화면
입력 항목
- 제목
- 고민 내용
- 현재 기분
- 미래에게 할 질문

버튼
- 미래에게 보내기

### 5.3 AI 답장 화면
- 봉투가 열리는 간단한 애니메이션
- 편지 형식 답장 표시

아래 버튼
- 저장하기
- 공유하기
- 다시 생성

### 5.4 타임캡슐 화면
- 날짜 + 제목 목록
- 클릭하면 상세 편지 확인

### 5.5 마이페이지
- 총 편지 수
- 감정 요약
- 성장 기록 요약

---

## 6. 서비스 톤앤매너
- 말투: 다정하고 편안하게
- 문장: 어렵지 않게, 짧고 명확하게
- 금지: 공격적 표현, 불안 조장 표현

---

## 7. MVP(첫 출시) 범위
첫 버전에서는 아래만 먼저 만든다.

- 회원가입/로그인
- 고민 작성
- AI 답장 생성
- 편지 저장 및 다시 보기
- 감정 기록 간단 통계

나중에 추가
- 1년/5년/10년 후 예약 편지 알림 기능

---

## 8. 기대 효과
- 고민을 글로 쓰며 마음이 정리된다.
- "미래 시점" 조언으로 불안이 줄어든다.
- 시간이 지나며 내 감정과 성장 변화를 볼 수 있다.

---

## 9. 예시 답장
To. 2026년의 나

안녕.
나는 2036년의 너야.
지금은 많이 흔들리겠지만, 너는 생각보다 잘 버티고 있어.
완벽한 답을 당장 찾지 못해도 괜찮아.
이번 주에는 네가 할 수 있는 작은 행동 하나만 해보자.
그 작은 선택이 너를 원하는 방향으로 데려갈 거야.

늘 응원하는,
10년 후의 너

---

## 10. 한 줄 정리
Future Letter는 "기록 + 위로 + 회고"를 한 번에 제공하는 감성 자기성찰 서비스다.
